개요

html 요소가 브라우저에 의해 어떻게 보여질지 결정하는 속성

웹 페이지의 레이아웃을 정할 때 자주 사용하는 속성

대부분의 html 요소는 block 또는 inline 중 하나로 결정되어 브라우저에 보여짐

하지만, 웹 페이지 레이아웃 구성 시 요소의 속성을 바꿔야 할 때가 존재하는데

이 때 display 속성으로 바꿀 수 있음
