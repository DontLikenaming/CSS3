CSS position 관련

개요

지금까지는 margin/padding 또는 인라인/블록 요소의 특성에 의해
부분적으로 요소의 위치를 지정할 수 있었음.

반면, position이라는 속성을 이용하면 html 요소들의 위치를 좌표등올 결정할 수 있음.

position은 static, absolute, relative, fixed, sticky, z-index 등의 속성값을 가지며,
그 후, top/left/right/bottom의 속성을 부여해서 해당 속성값만큼 이동할 수 있음.

static position

기본적인 요소 배치순서에 따라 위->아래, 왼쪽->오른쪽으로 배치되며,
부모-자식 관계의 요소에서 자식요소는 부모요소에 따라 배치됨.

absolute position

조상요소(<body> 태그) 또는 부모요소를 기준으로 속성값만큼 요소를 이동시켜 배치됨.

단, 부모요소를 기준으로 하위요소를 배치하려면 부모요소의 position은 relative여야 함.

fixed position

뷰포트(웹페이지 가시영역)를 기준으로 top/left/right/bottom 만큼 요소를 이동시켜 배치.

브라우저를 스크롤 하더라도 뷰포트의 특정위치에 요소가 배치됨.

relative position과 fixed position을 적절히 조합해서 만든 position.

z-index position

요소를 배치하다보면 설정된 위치나 방식에 따라 서로 겹칠 수 있음

기본적으로는 먼저 만들어진 요소가 나중에 만들어진 요소보다 밑에 배치됨

요소가 쌓이는 순서를 직접 지정하여 필요에 맞게 사용 가능함

z-index 값이 클수록 앞쪽에 배치됨
